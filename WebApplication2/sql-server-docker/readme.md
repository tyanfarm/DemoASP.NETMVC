# Chạy SQL Server trên Docker
(chạy lệnh ở thư mục chứa docker-compose.yml)
## Chạy dịch vụ
``````
docker-compose up -d
``````
## Khởi động nếu dịch vụ bị dừng
``````
docker-compose start
``````
## Xóa dịch vụ
``````
docker-compose down
``````
## Kiểm tra container đang chạy
``````
docker ps
``````
